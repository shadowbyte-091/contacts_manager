# Contact Manager (CLI Application)

## Project Overview
This project is a simple **Command Line Contact Manager** written in **C**.  
It allows users to store and manage contacts using **arrays** and **CSV file storage**.

The application supports basic **CRUD operations**:
- Add a contact
- List all contacts
- View a contact by ID
- Update a contact
- Delete a contact

All data is persisted to a CSV file on the local machine.

---

## Features
- Command-line interface
- Persistent storage using CSV files
- Modular C codebase
- No structs, no dynamic memory allocation
- Handles up to 500 contacts
